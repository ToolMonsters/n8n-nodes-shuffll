
# n8n-nodes-shuffll

Official n8n community node for [Shuffll](https://shuffll.com) - AI-powered video creation platform.

## Installation

### Via n8n Community Nodes (Cloud & Self-Hosted)
1. Go to **Settings > Community Nodes**
2. Click **Install**
3. Enter `n8n-nodes-shuffll`

### Via npm (Self-Hosted only)
```bash
npm install n8n-nodes-shuffll
```

## Credentials

You need a Shuffll API Key:
1. Go to [Shuffll Dashboard](https://app.shuffll.com)
2. Navigate to **Profile > API Integration**
3. Copy your API Key
